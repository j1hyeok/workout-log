# workout-log
・何を作ろうと思ったか：
　日々ウェイトトレーニングをする時、自分が遂行した運動を記録するツールを作りたいと考えました。
 
・なぜか：
　いつもトレーニングを行う時に頭で覚えましたが、毎回のトレーニングの内容を記録し、随時その記録を確認することで成長の進捗を把握したいと考えたためです。
 
・生成AIとのやり取り（プロンプトの履歴や工夫したやり取りなど）：
　私が頭で思い描いたことを例を挙げながら具体的に説明しようと努力しました。また、改善したい点を段階的に改善していくことで
　じっくりですが明確に作りたいものの解像度を上げていきました。
 
・詰まったところとどう乗り越えたか：
　実際ツールを起動してみてから使用する上で不便なことが何でどう改善するかを言語化することが大変でした。
　これを克服するために常に使用する側の視点に立って改善点を考えました。
　また改善のための仮説を生成AIに伝え、コーディングしてもらい、繰り返して改善点を模索しました。
 
・次にやるなら何を変えるか：
　入力した運動種目の間の修正をスムーズにできるようにし、スマートフォンを使用する背景を軸により使いやすいUIに改善したいと思います。
